# Audio Files

Place the following audio files in this directory:

## Music (loops in background)

- `music.mp3` - Background ambient music track (should loop seamlessly)

## Junction Sounds (played when entering a junction)

- `junction1.mp3` - Sound effect for junction entry
- `junction2.mp3` - Alternative junction sound

## Hallway Ambient Sounds (randomly played while walking)

- `ambient1.mp3` - Random ambient sound (footsteps, creaks, etc.)
- `ambient2.mp3` - Random ambient sound
- `ambient3.mp3` - Random ambient sound

## Recommendations

For a creepy maze atmosphere, consider:

- **Music**: Low droning ambient music, unsettling tones
- **Junction sounds**: Distant whispers, door creaks, subtle chimes
- **Ambient sounds**: Footsteps echo, distant thuds, breathing, wind
